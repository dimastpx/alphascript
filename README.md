> Принципиально новый язык программирования на русском языке!
> 
> "Давид Сурдул"

## Инструкция
### Создание переменных
Это очень простой процесс
```alphascript
переменная а
переменные б в г
```
Готово! переменные созданы! Но как сделать чтоб они были чему то равны?
Всё просто!
```alphascript
сделать а числом 10
сделать б строкой Привет
сделать в логикой да
сделать г суперчислом 10.5
```
